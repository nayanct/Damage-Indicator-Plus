# Damage Indicator Plus

Floating damage numbers for Minecraft — like Fortnite's damage tickers, fully customizable.

This repository is for **bug reports, feature requests, and general feedback**. The mod's source code is not public.

## How to Report a Bug or Request a Feature

We use GitHub **Issues** to track bugs and suggestions. If you've never used GitHub before, don't worry — here's how:

### Step 1: Make a GitHub account

If you don't have one yet, go to [github.com/signup](https://github.com/signup) and create a free account. You just need an email.

### Step 2: Go to the Issues tab

Click the **Issues** tab at the top of this page, or [click here](../../issues).

### Step 3: Check if someone already reported it

Before creating a new issue, scroll through the existing ones or use the search bar to see if someone already posted about the same thing. If they did, you can add a thumbs up reaction or leave a comment instead of making a duplicate.

### Step 4: Create a new issue

Click the green **New Issue** button. You'll see a template to fill out. Pick the right one:

- **Bug Report** — Something is broken or not working right
- **Feature Request** — You have an idea for something new

### Step 5: Fill it out

**For bugs**, include:
- What happened
- What you expected to happen
- Your Minecraft version (1.20.1 or 1.21.1)
- Your Fabric Loader version
- Any other mods you have installed
- A screenshot or video if possible (you can drag and drop images directly into the text box)

**For feature requests**, just describe what you want and why it would be useful.

### Step 6: Submit

Click **Submit new issue** and you're done. We'll respond when we get to it.

## Frequently Asked Questions

**Where do I download the mod?**
From CurseForge or Modrinth. Links are at the top of this page. Do not download from any other site.

**Do I need to install anything on the server?**
No. Damage Indicator Plus is client-side only. Just install it on your client.

**The numbers aren't showing up.**
Make sure the mod is enabled (press H to open settings and check the toggle). Also confirm you have Fabric API installed.

**I'm getting a crash on startup.**
Open a bug report and include your crash log. You can find it in `.minecraft/crash-reports/` or `.minecraft/logs/latest.log`.

**Can I include this mod in my modpack?**
No. The mod is All Rights Reserved. Contact us first.
